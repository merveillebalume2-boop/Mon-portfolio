## À propos
Portfolio professionnel développé avec HTML, CSS et JavaScript. Design moderne, responsive et interactif.

##  Fonctionnalités
-  Design responsive (Mobile, Tablet, Desktop)
-  Animations CSS et JavaScript
-  Navigation fluide avec scroll spy
-  Mode clair/sombre
-  Formulaire de contact
-  Portfolio interactif
-  Préloader d'animation
-  Effets de hover et transitions

## Technologies utilisées
- HTML5
- CSS3 (Flexbox, Grid, Animations)
- JavaScript (ES6+)
- FontAwesome Icons
- Google Fonts (Poppins)
- Typed.js pour l'animation texte

##  Compatibilité
- Chrome
- Firefox
- Safari
- Edge
- Mobile & Tablet
